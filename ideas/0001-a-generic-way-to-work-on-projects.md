# Idea 0001: A generic way to work on projects

- Status: promoted
- Date: 2026-06-28
- Parent: (none — this is the root idea)

## Idea

Find a *reasonable, structured, reproducible, economic way to work* on a
project — and make that way **generic**, so it is reusable for any project
rather than bound to a single one.

## How it arose (two shifts)

1. Started on a concrete project (**my-second-brain**) and wanted a better way
   to work on it.
2. Realized that the way-of-working should itself be generic — reusable for any
   project, not just that one.

So the subject became *"a generic way to work on projects."* It is its own
idea, developed concept-first. my-second-brain becomes simply the first project
that will later adopt the method.

A quiet irony: this method, fully formed, is exactly the thing that would tell
us how to develop ideas like itself. It is its own first subject.
